# Quasar App Extension QPdfviewer

QPdfViewer allows you to have PDF in your Quasar web pages.

[![npm](https://img.shields.io/npm/v/@quasar/quasar-app-extension-qpdfviewer.svg?label=quasar-app-extension-qpdfviewer)](https://www.npmjs.com/package/@quasar/quasar-app-extension-qpdfviewer)
[![npm](https://img.shields.io/npm/dt/@quasar/quasar-app-extension-qpdfviewer.svg)](https://www.npmjs.com/package/@quasar/quasar-app-extension-qpdfviewer)

## Everything you need for a complete PDF solution.

Remember, app extensions can _only_ be used with the Quasar CLI.

# Install
```bash
quasar ext add @quasar/qpdfviewer
```
Quasar CLI will retrieve it from NPM and install the extension.

# Uninstall
```bash
quasar ext remove @quasar/qpdfviewer
```

# Describe
You can use `quasar describe QMarkdown` for the QMarkdown component
